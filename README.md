# WRAITHEX

> Move like a wraith. Leave no trace.

A live cryptocurrency market dashboard with real-time price feeds, privacy scoring, and gold comparison. Built privacy-first, runs at $0.

## Features

- **Live prices** — 13 coins via CoinGecko (60s cache)
- **Real-time BTC & XMR ticker** — Bitfinex WebSocket, ticks on every trade
- **Gold vs Crypto panel** — PAXG live gold price, portability and privacy comparison
- **Privacy scoring** — XMR vs BTC, ETH, ZEC across 5 technical metrics
- **XMR donation button** — untraceable support
- Search, filter (gainers/losers), sort (rank/price/24h/mcap)

## Stack

- Next.js 14
- CoinGecko free API (no key needed)
- Bitfinex WebSocket (no key needed)
- PAXG via CoinGecko for gold price
- Vercel (free hosting)

**Total cost: $0. Total trackers: 0.**

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deploy

Push to GitHub, import to Vercel. Set root directory if needed.
